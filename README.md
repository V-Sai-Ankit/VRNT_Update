# Veda Rakshana Nidhi Trust (VRNT) Prototype

A modern, responsive frontend prototype for the Veda Rakshana Nidhi Trust, dedicated to the preservation and protection of ancient Vedic heritage.

## Features

- **Elegant Design**: Polished UI with thoughtful typography and visual depth.
- **Vedas Section**: Dedicated sub-page for exploring Vedic knowledge, featuring:
  - About the Vedas
  - Veda Rakshanam (The Vedic Tree)
  - What we owe to Veda Vyasa
- **Interactive Navigation**: Accordion-based organization for complex content.
- **Trust Information**: Detailed sections for Mission, Activities, Trustees, and News.
- **Contribution Portal**: Mockup for donation schemes and support options.
- **Responsive Layout**: Fully optimized for mobile and desktop viewing.

## Tech Stack

- **React**: UI Framework
- **Tailwind CSS**: Styling
- **Framer Motion**: Animations and transitions
- **Wouter**: Client-side routing
- **Lucide React**: Iconography
- **Shadcn UI**: Accessible components

## Project Structure

- `client/src/components/sections`: Reusable UI sections for the home page and beyond.
- `client/src/pages`: Individual page components (News, Trustees, Contact, Vedas, etc.).
- `client/src/lib/constants.ts`: Centralized configuration for site content.

## Mission

To strive every nerve to protect and preserve our holy Vedic heritage and pass it on to posterity.
